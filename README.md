# Salt Guide Viewer

Map viewer for [Salt Guide](https://saltguide.com.au/) weekly fishing spots across Port Phillip Bay, Western Port, and beyond. Filter by fish type, region, and week.

![screenshot](screenshot.png)

**[View live](https://massiminoe.github.io/salt-viewer/)**

## Usage

To update with new data, drop `.txt` report files into `data/` and run:

```
python3 parse.py
```
